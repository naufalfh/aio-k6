# aio-k6
k6 playground

Get Started:
- install k6
- init a project

How to use:
- Run the test: 
    k6 run script.js
- Run the test with k6 Dashboard
    - cmd/terminal: K6_WEB_DASHBOARD = true k6 run script.js;
    - using poweshell: $env:K6_WEB_DASHBOARD = "true"; k6 run script.js;

k6 Dashboard:
http://127.0.0.1:5665/
